# Twitter Sentiment Analysis
**Web App: https://twitter-sentiment-prediction.streamlit.app/**

**Goal:**
- Predict sentiment of a tweet using NLP
- Visualize insight from twitter sentiment analysis
- Train model using 1.6 million tweets
- Get high accuracy model and compare each model
- Deploy the best model into web app
- User can input their own tweet and get the sentiment of the tweet

**Model:**
- Logistic Regression 
- Bernoulli Naive Bayes
- XGBoost Classifier
